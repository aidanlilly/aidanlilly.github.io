Maze Generation

Depth-First Search

Made using Java Open Processing. This program starts by generating a grid. Starting from the 0th index, or top left, it randomly selects a direction to go. It then proceeds one cell in that direction removing the wall between its current cell and the new one. The new cell is then appended to a queue and marked as visited. This process will continue until the program becomes trapped at a location as all neighbouring cells have been visited. To proceed, it will begin backtracking through the queue, testing for unvisited neighbouring cells. This process will continue until it backtracks to the 0th index with no neighbouring cells remaining. Once completed, the user will then be given control of a character that they can guide through the newly generated solvable maze.


Swing Phsyics

Vector Math

Made with Unreal Engine 4. This program allows the user to swing through a 3D city. The environment and swing mechanics are inspired by Japanese anime "Attack on Titan." The game functions within a loop. Each iteration of that loop, the players location in the 3D space is stored as a 3-vector. The forward facing vector is then drawn from the player to its forward facing position, in other words wherever the red dot is located. When the user right clicks, the program checks if the dot has made contact with an actors hitbox within a specified range. If this is the case, then a vector is drawn from the actor to the user. Each proceeding iteration draws the user at the head of that vector. As well, its magnitude is decrementing resulting in the character pulling towards it causing the forward force. To simulate the swinging mechanic, the player is also acting on by a gravity vector simulating a downward force.

Machine Learning

Gradient Descent

Made in Python. This linear regression model will use the square error function to predict the value of any input within a dataset according to other values with simlar features. This program is limited to two dimensions as it is for presentation purposes but it can be expanded into an infinite with simple modifications. The main purpose of the program is to create a line of best fit through a randomized dataset which can then be used to predict the place of any given value among the known values.
